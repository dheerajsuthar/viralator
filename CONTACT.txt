The Viralator Mailing List
The Viralator mailing list is the place to post questions, ask for features or give some hints to the development team. This is an English mailing list and you need to subscribe to it first before posting messages.

Before subscribing to the mailing list, please read these general guidelines to have more information about how to using it.

For more information about the mailing list: http://groups.yahoo.com/group/viralator.

To post messages send an email to: viralator@yahoogroups.com
To subscribe send an email to: viralator-subscribe@yahoogroups.com
To unsubscribe send an email to: viralator-unsubscribe@yahoogroups.com
To contact the list owner: viralator-owner@yahoogroups.com
